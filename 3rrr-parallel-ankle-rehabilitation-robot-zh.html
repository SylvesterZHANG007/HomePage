从结构设计到动力学建模：面向控制的3-RRR并联踝关节康复机器人开发

arXiv预印本：2024年5月
康复机器人 并联机构 控制系统 可穿戴设备

研究概览： 本研究提出了一种基于3-RRR球面并联机构的可穿戴踝关节康复机器人的全面开发，包含从结构设计到运动学建模，再到基于拉格朗日的动力学建模和控制仿真的完整设计至动力学流水线。

研究背景

踝关节损伤在运动员、术后患者以及神经功能障碍患者中十分常见。有效的康复训练对于恢复移动能力、力量和本体感觉至关重要。然而，传统康复设备往往只提供有限的自由度，限制了自然关节运动，可能延长康复时间。

人体踝关节在生物力学上可近似为具有三个旋转自由度的球关节：俯仰（跖屈/背屈）、滚转（内翻/外翻）和偏航（内旋/外旋）。尽管人体真实的几何模型对应更复杂的多自由度运动，但是由于关节的运动可以都放在同一个包络面内，所以可以近似的用球面模型来表达脚踝关节。为了复制这种自然运动，我们提出将3-RRR球面并联机构（SPM）作为可穿戴踝关节康复机器人的核心。

图1：人体踝关节解剖结构（左）显示了复杂的骨骼结构和关节排列，以及在球面包络内近似踝关节运动的3-RRR球面并联机构设计（右）。

机构设计与工程考量

3-RRR球面并联机构由三个相同的分支组成，每个分支有两个旋转关节（R）连接固定基座和运动平台。这种配置能够实现紧凑、对称的结构，能够围绕共同的旋转中心产生三个旋转自由度。

设计理念

该设备专为直接下肢安装而设计，促进日常使用并提高患者依从性。主要设计考量包括：

紧凑型基环： 配备可调节固定带，可安全、舒适地固定在小腿上。

无干扰机构： 低轮廓分支和关节外壳最大限度地减少对相邻关节的阻碍，确保与鞋类和支具的兼容性。

设备模块化： 可互换连杆和快拆紧固件支持跨用户和康复阶段的调整。

图2：SolidWorks CAD模型，展示了针对工作空间、紧凑性和便携性优化的设计。

运动学分析与运动规划

运动学分析建立了平台姿态与各关节运动之间的映射关系。我们首先定义所有符号，然后比较坐标表示，推导坐标系变换，最后给出正逆运动学和速度运动学。这些结果为下一节的动力学建模奠定基础。

几何框架

3-RRR机构由三个相同的运动链组成，每个运动链通过主动关节（\(A_i\)）和被动关节（\(B_i\)）将基座平台与足部平台连接。这种配置在保持结构刚性的同时实现了三个旋转自由度。

运动学映射

系统使用欧拉角变换实现平台方向与关节角度之间的映射：

\[C_i = R(\alpha,\beta,\gamma)\,C_{i0}\]

其中\(\alpha\)、\(\beta\)和\(\gamma\)表示踝关节的滚转、俯仰和偏航旋转。

正逆运动学

运动学解决方案实现以下双向映射：

  • 逆运动学： 给定期望踝关节姿态 → 计算所需驱动器角度\(\theta_i\)
  • 正运动学： 给定驱动器角度\(\theta_i\) → 确定结果踝关节姿态

速度关系由系统雅可比矩阵控制：

\[\dot{\mathbf{q}} = J_r\,\omega\]

其中\(J_r\)将关节速度与平台角速度相关联，对平滑运动控制至关重要。

动力学建模与控制框架

开发了全面的基于拉格朗日的动力学模型，以实现精确的扭矩估计和仿真分析。控制框架集成了具有可调顺应性特征的PID控制器，以适应不同的康复要求。

动力学模型

系统动力学由基于拉格朗日的运动方程描述：

\[M(\mathbf{q})\,\ddot{\mathbf{q}} + C(\mathbf{q},\dot{\mathbf{q}})\,\dot{\mathbf{q}} = \boldsymbol{\tau}\]

其中\(\mathbf{q} = [\theta_1, \theta_2, \theta_3]^T\)表示驱动器角度，\(M(\mathbf{q})\)是质量矩阵，\(C(\mathbf{q},\dot{\mathbf{q}})\)捕获科里奥利效应，\(\boldsymbol{\tau}\)是施加的扭矩。

控制框架

系统采用独立的PID控制器对每个旋转自由度（滚转、俯仰、偏航）进行控制，实现精确的运动跟踪和可调节的顺应性特征。

每个轴遵循控制律：

\[\tau_i = K_{p,i}\,e_i + K_{d,i}\,\dot{e}_i + K_{i,i} \int e_i\,dt\]

其中\(e_i = q_{i,\text{ref}} - q_i\)是每个轴的跟踪误差。

自适应顺应性

控制器增益可以调节以提供不同的顺应性行为：

  • 柔性辅助： 较低增益用于早期康复阶段的温和、顺应性支持
  • 精确控制： 较高增益用于主动治疗练习中的准确跟踪

这种适应性使同一设备能够适应患者康复的不同阶段和治疗要求。

仿真结果与验证

初步仿真验证了在代表性康复轨迹下的稳定关节协调和平滑运动跟踪。系统在不同顺应性设置下表现出稳健的性能，能够为个体患者需求进行定制。

关键发现： 尽管3-RRR SPM在机械上是刚性的，但调节反馈增益可有效调节其顺应性特征，使同一设备能够根据康复要求提供温和辅助或精确控制。

临床应用与治疗效益

3-RRR并联踝关节康复机器人解决了传统康复设备的几个局限性：

多自由度自然运动

与传统单轴设备不同，该系统能够同时进行俯仰、滚转和偏航运动，这些运动密切复制自然踝关节生物力学，促进更有效的康复结果。

个性化治疗方案

可调节的顺应性特征允许临床医生根据个体患者进展和特定康复目标定制治疗强度和辅助水平。

可穿戴形态因子

紧凑、轻便的设计促进患者依从性，并支持临床和家庭康复场景，扩展治疗可及性。

未来发展与研究方向

基于已建立的基础，正在开发几个先进功能：

自适应控制框架

集成鞋内IMU和足底压力传感与实时模型个性化，将实现对个体患者生物力学和进展的动态适应。

步态相位集成

模糊逻辑阻抗控制器将围绕动态估计的关节轴运行，并根据步态相位切换模式，实现从静态关节活动度训练到步态定时辅助的平滑过渡。

增强生物力学建模

未来设计迭代将整合双轴、非同心踝关节模型，以更好地匹配人体生物力学并改善人体工程学舒适性，超越当前的单一同心旋转中心假设。

更广泛的影响与应用

除了临床康复，该平台还可重新配置为神经肌肉功能障碍个体的辅助行走或跑步设备，提供实时平衡矫正和动态支持以增强移动能力和生活质量。

研究影响： 本研究为智能、个性化踝关节康复建立了基础平台，既支持静态训练又可扩展到步态相位定时辅助，同时展示了如何将并联机构设计优化用于可穿戴治疗应用。

结论

本研究提出了可穿戴3-RRR踝关节康复机器人的全面设计至动力学工作流程，从便携式机械设计到运动学和动力学建模再到基于仿真的控制。系统成功展示了如何将并联机构原理适应可穿戴治疗应用，同时保持有效康复所需的精度和控制特性。

经过验证的平台为推进个性化踝关节康复技术提供了坚实基础，正在进行的开发专注于自适应控制、步态集成和增强生物力学建模，以进一步改善治疗结果和患者体验。
